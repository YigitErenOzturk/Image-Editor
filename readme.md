# Simple Image Transformation Editor

## Overview
This project is a multimedia application that allows the user to load an image, apply selected image transformations, preview before/after, and export the result.

## Functional Requirements
- FR1: Load image from disk
- FR2: Apply at least 4 transformations (Grayscale, Blur, Edge, Brightness/Contrast)
- FR3: Preview before/after
- FR4: Export processed image

## Tech Stack
- Python 3
- OpenCV
- Pillow
- Tkinter
